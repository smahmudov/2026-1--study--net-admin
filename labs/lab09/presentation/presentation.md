# Теоретическая часть

## Назначение STP

* **STP (Spanning Tree Protocol)** предотвращает образование петель в сети второго уровня
* При наличии избыточных соединений отключает лишние каналы логически
* Обеспечивает корректную работу сети при сохранении резервных путей
