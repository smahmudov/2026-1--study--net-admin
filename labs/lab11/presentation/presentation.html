Теоретическая часть

Что такое NAT

  • NAT (Network Address Translation) — технология преобразования IP-адресов
  • Частные адреса локальной сети заменяются на внешние при выходе в Интернет
  • Основные задачи:
    • экономия публичных IP-адресов
    • доступ локальной сети к внешним ресурсам
    • скрытие внутренней структуры сети

Виды NAT

  • Static NAT
    • постоянное соответствие одного внутреннего IP одному внешнему
  • Dynamic NAT
    • временное сопоставление адресов из пула
  • PAT / NAT Overload
    • множество внутренних узлов работают через один внешний IP за счёт портов
  • SNAT
    • изменяется адрес источника
  • DNAT
    • изменяется адрес назначения
