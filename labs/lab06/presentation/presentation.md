# Теоретическая часть

## Маршрутизация и инкапсуляция

* **Маршрутизация** — передача пакетов между разными сетями по IP-адресу назначения
* **Инкапсуляция** — добавление заголовков протоколов при прохождении данных по уровням модели **OSI**
* Маршрутизатор работает на **сетевом уровне**
* Коммутатор передаёт кадры на **канальном уровне**

## VLAN и IEEE 802.1Q

* **VLAN** позволяет логически разделять сеть на отдельные сегменты
* **IEEE 802.1Q** добавляет тег VLAN в Ethernet-кадр
* Один физический канал может передавать трафик нескольких VLAN
* Для этого используется **trunk-порт**

## Схема Router-on-a-Stick

* Один физический интерфейс маршрутизатора обслуживает несколько VLAN
* На интерфейсе создаются логические подинтерфейсы
* Каждому подинтерфейсу назначаются:
  * номер VLAN
  * IP-адрес шлюза
  * инкапсуляция **dot1Q**

* При прохождении через маршрутизатор меняются **MAC-адреса**
* **IP-адреса** источника и назначения сохраняются

* Межвлановый обмен выполняется корректно
* Тегирование **802.1Q** позволяет передавать трафик нескольких VLAN по одному каналу
