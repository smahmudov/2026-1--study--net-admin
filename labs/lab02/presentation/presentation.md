# Цель работы

## Основная цель

* Получить навыки начальной настройки сетевого оборудования Cisco
* Изучить способы подключения к маршрутизатору и коммутатору
* Освоить настройку удалённого доступа (Telnet/SSH)
* Проверить работоспособность сети

# Теоретическая часть

## Способы подключения

* Консольный кабель — первичная настройка
* Telnet — удалённый доступ без шифрования
* SSH — защищённый удалённый доступ
* AUX — подключение через модем
