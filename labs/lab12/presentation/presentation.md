# Теоретическая часть

## Что такое NAT

* **NAT (Network Address Translation)** — преобразование IP-адресов
* Частные адреса → публичный IP
* Выполняется на маршрутизаторе

## Виды NAT

* **Static NAT** — 1:1 соответствие
* **Dynamic NAT** — адрес из пула
* **PAT (overload)** — один IP, разные порты
