Теоретическая часть

Статическая маршрутизация

  • Маршруты задаются администратором вручную
  • Используется в сетях с простой и предсказуемой топологией
  • Позволяет явно определить путь передачи трафика
  • Часто применяется для:
    • подключения филиалов
    • маршрута по умолчанию
    • лабораторных и учебных сетей

Маршрутизация между VLAN

  • Каждый VLAN относится к отдельной IP-подсети
  • Для каждого VLAN настраивается шлюз по умолчанию
  • Передача трафика между VLAN выполняется через маршрутизатор или L3-коммутатор
  • Для удалённых VLAN-сетей могут использоваться статические маршруты
