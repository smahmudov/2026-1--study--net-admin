# Теоретическая часть

## Hub vs Switch vs Router
* **Hub (концентратор, L1)**
  * Повторяет сигнал на все порты
  * Общий домен коллизий → возможны коллизии
* **Switch (коммутатор, L2)**
  * Пересылает кадры по **MAC-таблице**
  * Разделяет домены коллизий (обычно full-duplex)
* **Router (маршрутизатор, L3)**
  * Соединяет разные IP-сети
  * Маршрутизирует по IP-адресам

## Служебные протоколы (в работе)
* **ARP**: IP → MAC
* **ICMP**: диагностика (ping / Echo)
* **STP**: предотвращение петель в L2
* **CDP**: служебный протокол Cisco для информации о соседях
